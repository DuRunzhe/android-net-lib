# How To Use
Add it to your project build.gradle with:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
and your module build.gradle:

```gradle
dependencies {
    implementation 'com.github.DuRunzhe:android-net-lib:v1.0.2-alpha'
}
```
